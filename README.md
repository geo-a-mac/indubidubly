## Indubitably

Indubitably is a web based app designed to connect Job Seekers and Employers through skills alone.

### Technologies
- HTML
- CSS
- Bootstrap
- Handlebars
- Javascript
- Sequelize
- Express
- MySql2
- Connect-session-sequelize
- Express-session
- Argon2
- Heroku
